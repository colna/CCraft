# DevChat — 交互设计文档

## 1. 信息架构

DevChat 包含两个用户界面：主项目 App 面向移动端对话式开发，官网 App 面向产品展示、下载转化和文档入口。

### 1.1 主项目 App

```
App
├── 🏠 首页（最近项目 + 快捷入口）
├── 📁 项目（GitHub Repo 列表）
├── 💬 对话（AI 编程主界面）  ← 核心
├── 📋 历史（会话记录）
└── ⚙️ 设置（AI配置 / GitHub / 偏好）
```

### 1.2 官网 App

```
Website
├── 首页（产品定位 + 关键能力 + 下载入口）
├── 功能（对话开发 / Diff 审查 / 安全存储 / GitHub 集成）
├── 下载（iOS / Android / 桌面端占位）
├── 更新公告
└── 文档与支持（快速开始 / FAQ / 联系方式）
```

## 2. 页面流程

### 2.1 主项目 App 首次使用流程

```
启动 App
  → 欢迎页（简介 3 页滑动）
  → 配置 AI 模型
     ├── 输入 Base URL
     ├── 输入 API Key
     ├── 选择/输入 Model Name
     └── 点击「测试连接」→ 成功提示
  → 绑定 GitHub
     ├── 点击「连接 GitHub」
     ├── OAuth 网页授权
     └── 返回 App，显示授权成功
  → 进入首页
```

### 2.2 主项目 App 核心开发流程

```
选择项目
  → 加载项目快照
     ├── 首次：显示「正在分析项目结构...」进度
     └── 非首次：秒级加载缓存
  → 进入对话界面
  → 用户输入需求（文字）
  → AI 流式返回响应
     ├── 文字说明部分：实时渲染
     └── 代码 Diff 部分：卡片形式展示
  → 用户查看 Diff
     ├── 点击 Diff 卡片 → 全屏 Diff 查看
     ├── 文件列表 → 逐个查看
     └── 不满意 → 继续对话修改
  → 满意后点击「提交变更」
     ├── 确认修改文件列表（可勾选/取消）
     ├── 编辑 Commit Message
     ├── 选择分支
     └── 点击「Commit & Push」
  → 提交成功 ✓
```

### 2.3 官网访问流程

```
进入官网首页
  → 查看产品定位与核心能力
  → 点击下载入口
     ├── 移动端：跳转对应应用商店或安装说明
     └── 桌面端：展示平台状态与等待名单入口
  → 需要了解更多
     ├── 查看功能详情
     ├── 查看更新公告
     └── 进入文档与支持
```

## 4. 交互细节

### 4.1 对话交互

| 交互 | 行为 |
|------|------|
| 发送消息 | 显示 loading 动画，AI 流式输出文字 |
| Diff 卡片 | 可点击展开内联预览，或点击全屏查看 |
| 长按消息 | 复制 / 重新生成 / 删除 |
| 引用文件 | 输入 `@` 触发文件选择器，引用特定文件 |
| 上滑加载 | 加载更早的对话历史 |

### 4.2 Diff 交互

| 交互 | 行为 |
|------|------|
| 左右滑动 | 切换文件 |
| 点击行号 | 展开/折叠上下文 |
| 勾选文件 | 选择要提交的文件 |
| 双指缩放 | 调整代码字号 |
| 语法高亮 | 根据文件类型自动高亮 |

### 4.3 提交交互

| 交互 | 行为 |
|------|------|
| Commit Message | AI 自动生成，用户可编辑 |
| Push 按钮 | 需二次确认（防误触） |
| 提交成功 | 显示成功动画 + commit hash |
| 提交失败 | 显示错误原因 + 重试按钮 |

### 4.4 手势操作

| 手势 | 位置 | 行为 |
|------|------|------|
| 右滑 | 对话页 | 返回项目列表 |
| 下拉 | 项目列表 | 刷新仓库列表 |
| 长按 | 项目卡片 | 显示快捷菜单（刷新快照/删除缓存） |

## 6. 状态与反馈

### 6.1 加载状态

| 场景 | 反馈方式 |
|------|----------|
| 项目快照生成中 | 进度条 + 当前步骤文字（"分析目录结构..."） |
| AI 回复中 | 流式文字 + 光标闪烁动画 |
| Push 中 | 全屏 loading + 进度提示 |
| Repo 列表加载 | 骨架屏（Skeleton） |

### 6.2 空状态

| 场景 | 展示 |
|------|------|
| 无最近项目 | 插画 + "选择一个项目开始编程" 按钮 |
| 无历史会话 | "还没有对话记录" |
| 搜索无结果 | "没有找到匹配的仓库" |

### 6.3 错误处理

| 错误类型 | 处理方式 |
|----------|----------|
| AI 连接失败 | Toast 提示 + 跳转设置页 |
| GitHub Token 过期 | 弹窗提示重新授权 |
| Push 冲突 | 显示冲突详情，建议用户在电脑上解决 |
| 网络断开 | 底部 Banner 提示，恢复后自动重连 |
