## 安全说明

- 前端仅展示 masked API Key，不保存或读取密钥明文。
- 设置页输入的 Claude API Key 只通过 `save_secret` 发送给 Rust 侧，随后清空本地输入状态。
- Tauri secret 读写集中在 Rust command 中；AI 测试连接只接收 `apiKeySecretRef`，实际 key 由 Rust 侧读取。
- 官网仅承载公开内容，不接入用户密钥、GitHub Token 或项目快照。
- 当前 GitHub 网络能力为 MVP adapter；Claude 连接测试已接入真实 Anthropic Messages API 形态。

## 外部来源核对

- Anthropic Messages API 使用 `POST /v1/messages`，请求包含 `model`、`max_tokens`、`messages`，并使用 `x-api-key` 与 `anthropic-version` 请求头。
- Anthropic Models API 使用 `GET /v1/models`，本次测试 endpoint 也返回 HTTP 200。
- 本报告不记录明文 API Key；提交前已扫描仓库，未发现用户提供的测试 key。

## 2026-05-11 官网重构

### 本次范围

- 将 `apps/web` 官网重构为 English-first bilingual 单页官网，中文作为辅助文案。
